Данный скрипт работает с таким текстом html тэга, значение которого есть item.name.
При наведении указателя мыши на объект скрипт подгружает данные с файла json, сравнивает ключ со именем объекта item.name и, если они совпадут, то к этому объекту выводится тайтл (подсказка) с данными, заложенными в значение ключа.
